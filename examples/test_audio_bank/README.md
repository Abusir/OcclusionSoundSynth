# Example Audio Bank

This directory contains a tiny smoke-test audio manifest for the public examples.

The WAV files are selected from ESC-50 and are included only as small real-world examples for running the SoundSpaces/Habitat-Sim synthesis pipeline.

Source dataset:

- ESC-50: Dataset for Environmental Sound Classification
- Original repository: https://github.com/karolpiczak/ESC-50
- License: Creative Commons Attribution-NonCommercial 3.0 Unported

The original ESC-50 filenames are recorded in `manifest.csv` under `source_filename`.

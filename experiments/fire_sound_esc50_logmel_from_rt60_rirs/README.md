# ESC-50 Crackling Fire Log-Mel Demo

This experiment renders `git_version/examples/test_audio_bank/esc50_crackling_fire.wav` through the six-scene RIRs generated by the corrected semantic-material pipeline.

The source audio is converted to mono, resampled to 16 kHz, zero-padded to 10 s, and convolved with each scene RIR. Time-frequency figures use log-Mel spectrograms instead of STFT spectrograms:

- source log-Mel: `results/figures/fire_source_10s_logmel.png`
- receiver log-Mel: `results/figures/*_fire_receiver_logmel.png`
- receiver-source log-Mel difference: `results/figures/*_fire_receiver_minus_source_logmel.png`

Run `bash run.sh` from this directory to regenerate the demo in `generated_soundspaces_runs/fire_sound_demo_esc50_logmel_zero_pad_from_rt60_rirs`.
